The Diode

Main functions of a Diode
  • Allow current to pass in one direction, while blocking it in the reverse direction
  • Can be used as shot-noise generators
  • light-emitting diodes (LED's) can be used to generate light
  • Zener-Diodes are used to protect circuits from high voltage surges
Name Picture Function
Zener-Diode Used to protect circuits from high voltage surges
light-emitting diode (LED) Used to generate light
